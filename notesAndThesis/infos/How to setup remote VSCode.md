

### Sidenote: Remote working with VisualStudioCode

Another way to connect to our servers for remote working is Visual Studio Code (https://code.visualstudio.com/).

Visual Studio Code provides you with a File Explorer, an Editor and a Terminal to manage your files and program/run your code. 

As explained here (https://code.visualstudio.com/docs/remote/ssh) you can install the remote-SSH plugin to connect to ssh-servers.

With the remote-SSH pluging you can connect to our servers via the command palette with your username and host: `<username>@file.mip.informatik.uni-kiel.de`.

Alternatively, you can configure your .ssh config file so that `file` and even your workgroup pc are already known as targets and you just have to select them from a list in the command palette.
